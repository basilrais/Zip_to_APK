# 📦 ZIP to APK Converter App

A simple Android utility app that allows users to **convert ZIP files containing APKs into installable APKs** or extract APKs from ZIP archives. This makes APK distribution and installation easier for developers and teams.

---

## 📱 App Overview

Sharing APK files can sometimes be cumbersome. This app solves the problem by allowing:

- Directly **install APK from ZIP** in one click  
- **Extract APK** from ZIP for manual installation  
- Works only for ZIP → APK conversion (not APK → ZIP)

The application is built natively for Android using **Kotlin** and **XML layouts** in **Android Studio**, ensuring smooth performance and a lightweight user experience.

---

## ✨ Key Features

- 📥 **Direct Install from ZIP:** Install APKs stored in ZIP archives in one click  
- 📂 **Extract APK from ZIP:** Extract APKs for manual installation or sharing  
- ⚡ **Fast & Lightweight:** Optimized for speed and minimal storage usage  
- 🧩 **Simple UI:** Clean interface for easy usage
